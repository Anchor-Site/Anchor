A Simple Blog Post #1

04/23/2018

This is a pretty awesome website. It's editable in the browser using Anchor, a client-side static site generator. Don't know what a static site generator is? Doesn't matter. It's cool, it's hip, and it makes working with your website 110% easier. Try it. You can do cool stuff like set text bold using CTRL-B, or make it underlined using CTRL-U. You can probably figure out how to make it italics all on your own.
